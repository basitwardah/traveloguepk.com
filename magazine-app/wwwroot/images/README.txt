DECORATIVE:
- coffee-cup.png (Login modal decoration, optional)

NOTES:
------
- All images should be optimized for web (compressed)
- Use JPG for photos, PNG for graphics with transparency
- Maintain aspect ratios mentioned above
- If images are missing, the site will use placeholder images from Unsplash
- You can use your own images or download from free stock photo sites like:
  * Unsplash.com
  * Pexels.com
  * Pixabay.com
